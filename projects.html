    Due: May 13 by 11:59pm | Points: 10 | Submission: Website URL

    Description: This project involved creating an interactive web page where users can enter text and apply various styles dynamically using JavaScript. The "Fancify my Text" page allows users to:

    • Use buttons and radio buttons to apply different styles such as bold, underline, and changing text size.
    • Experiment with JavaScript event handlers and DOM manipulation to enhance UI interactivity.

    • Bigger button to increase text size.
    • Radio buttons to toggle between 'FancyShmancy' and 'BoringBetty' styles.
